asyncResult (IAsyncResult)
The IAsyncResult object representing the asynchonous operation, as returned by BeginRead(array<Byte>[]()[], Int32, Int32, AsyncCallback, Object).
Return Value
The number of bytes transfered during the operation.
Remarks
Every call to BeginRead(array<Byte>[]()[], Int32, Int32, AsyncCallback, Object) must have a matching call to EndRead(IAsyncResult) to dispose of any resources used and to retrieve the result of the operation. When the operation was successful the method returns the number of bytes that were transfered. If an error occurred during the operation this method will throw the exceptions that would otherwise have ocurred during the operation. If the operation is not yet finished EndWrite will wait for the operation to finish before returning.
